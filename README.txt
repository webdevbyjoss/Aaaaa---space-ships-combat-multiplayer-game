Aaaaa - multiplayer space ship combats game

- HTML5 Canvas
- Websockets
- PHP server on plain sockets

Light weight & fast PHP server and multiplayer system based on WebSockets were buid from scratch and designed to get a lowest possible latency.

Libraries used:
- http://phpws.googlecode.com/ For WeSockets frames encoding/decoding
- http://jsgamesoup.net/ Asteroids positions pseudo random generation and game world handing is based on avesome jsGameSoup engine
